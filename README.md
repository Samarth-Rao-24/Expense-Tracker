# Expense Tracker Web App

A Flask-based web application to track daily expenses with user authentication.

## Features
- User registration and login
- Add, edit, delete expenses
- Filter by date and category
- Category-wise summary
- Chart visualization
- Secure password hashing

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS
- Chart.js

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`

2. Create MySQL database  
   `expense_tracker`

3. Run the app  
   `python app.py`

4. Open browser  
   `http://127.0.0.1:5000`
